# Field Robots - Robotic Master Project

This repository contains a field robotics project developed for simulation and control of mobile robots in CoppeliaSim. The main goal is to test autonomous navigation, trajectory following, and data acquisition using Python and Lua scripts.

---
## Usage
```bash
python -m coppeliaSim.control.main [--mode plan] [--path_type min_energy] [--controller pure_pursuit]
```
> Use --help to show all available options

## Robot Used

The robot is a differential-drive field robot equipped with:
- Encoders for wheel odometry
- IMU for orientation and linear acceleration
- Python API interface for simulation control
- Lua scripts to interact with CoppeliaSim via remote API

## CoppeliaSim Folder

This folder contains all the scripts, utilities, and configuration files needed to run and control the robot in CoppeliaSim.

---

### `starters/`

Lua scripts used to initialize communication with the CoppeliaSim API.

**How to apply:**
1. Right-click on an object in the scene hierarchy (e.g., the robot or the floor).
2. Select **Add → Script → Simulation Script → Threaded**.
3. Paste the content of the desired starter script from `coppeliaSim/starters/`.

**Available scripts:**
- `api_connection_script.lua` – Establishes the connection between Python API and CoppeliaSim.  
- `check_objects_script.lua` – Verifies the presence and handles of objects in the scene.  
- `store_csv_script.lua` – Saves simulation data to CSV files.

---

### `test/`

Contains Python scripts to test API communication.

**Scripts:**
- `test_connection.py` – Verifies the communication between Python and CoppeliaSim.

---

### `control/`

Python scripts for controlling the robot within the simulation.

**Scripts:**
- `main.py` – Entry point for running the robot control loop.  
- `main_code.py` – Implements the main control algorithms (trajectory following, path planning, etc.).

---

### `utils/`

Utility Python modules for general and mathematical support.

**Modules:**
- `math.py` – Math functions for navigation and path calculations.  
- `utils.py` – General helper functions for control and simulation.

---

### `config.yaml`

Configuration file containing parameters for the robot, simulation, and controllers.
